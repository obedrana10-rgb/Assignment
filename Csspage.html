What is CSS?

CSS stands for Cascading Style Sheets. It is used to style and layout web pages, including colors, fonts, and spacing.

CSS separates content from design, making web development more efficient.

Basic CSS Syntax

selector {
  property: value;
}

The selector points to the HTML element you want to style. Inside the block, each property is followed by a value.

Ways to Add CSS

  • Inline CSS – directly in the HTML element using the style attribute.
  • Internal CSS – inside a <style> tag in the HTML <head>.
  • External CSS – in a separate file (like style.css) linked with <link>.

Examples of CSS

1. Changing Text Color

p {
  color: blue;
}

2. Styling Background

body {
  background-color: lightgray;
}

3. Adding Borders

div {
  border: 2px solid black;
}

4. Changing Fonts

h1 {
  font-family: Verdana, sans-serif;
  font-size: 28px;
}

5. Adding Padding and Margin

.box {
  padding: 20px;
  margin: 10px;
  background: white;
}

6. Hover Effect

a:hover {
  color: red;
  text-decoration: underline;
}
