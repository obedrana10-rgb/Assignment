What is JavaScript?

JavaScript is a high-level, versatile programming language that powers the interactive behavior of websites. While HTML structures the page and CSS styles it, JavaScript makes it come alive by:

  • Updating and changing both HTML and CSS.
  • Reacting to user actions like clicks, scrolls, or typing.
  • Fetching data from servers (AJAX / APIs).
  • Building full web applications (with frameworks like React, Angular, Vue).

Every modern browser has a built-in JavaScript engine (e.g., Chrome’s V8) to run scripts quickly.

Variables in JavaScript

Variables are used to store data values. In JavaScript, you can declare variables in three ways:

  • var – the old way (function-scoped, can be redeclared).
  • let – the modern way (block-scoped, can be reassigned).
  • const – block-scoped but cannot be reassigned (constant values).

Examples

var city = "Kathmandu";   // function-scoped
let age = 22;             // block-scoped
const PI = 3.14159;       // constant, cannot be changed

Variables can store different types of data, such as:

  • String: text data ("Hello")
  • Number: integers or decimals (25, 3.14)
  • Boolean: true/false values
  • Array: list of values ([1, 2, 3])
  • Object: key-value pairs ({name: "Obed", age: 25})

Operators

Operators are symbols used to perform operations on values and variables:

  • Arithmetic: +, -, *, /
  • Comparison: ==, ===, >, <
  • Logical: &&, ||, !
let a = 10, b = 5;
let sum = a + b;

Functions

Functions are reusable blocks of code that perform a specific task. They help avoid repetition and keep code organized.

function greet(name) {
  return "Hello, " + name + "!";
}

JavaScript Events

Events are actions that happen in the browser, like a user clicking a button, moving the mouse, or pressing a key. JavaScript can listen for these events and respond.

<button onclick="showDate()">Show Date</button>

JavaScript and the DOM

The DOM (Document Object Model) is a representation of the web page structure. JavaScript can access and manipulate the DOM to dynamically update content.

document.getElementById("demo").innerHTML = "Hello, JS!";

This text can be changed with JS.

More JavaScript Examples

Alert Example

The alert() function shows a popup box with a message.

function showAlert() {
  alert("Hello! This is a JavaScript alert.");
}

Console Example

You can use console.log() to print messages for debugging inside the browser console (press F12 to see).

function logMessage() {
  console.log("This message was logged using JavaScript.");
}
