What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language used to create web pages. HTML elements tell the browser how to display content.

HTML documents are made up of tags and attributes that define headings, paragraphs, links, images, and more.

Basic HTML Document Structure

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

This is the simplest HTML5 template. Every page starts with <!DOCTYPE html>, followed by the <html> element as the root.

Main Features of HTML

  • Defines the structure of a web page.
  • Supports text, images, links, videos, and forms.
  • Works with CSS and JavaScript to create interactive and styled pages.
  • Uses elements and tags enclosed in angle brackets (< >).
